# React Native

## How to install

_Clone Project:_

```sh
git clone https://github.com/abcprintf/rn-psc-new.git
```

_Setup with Expo:_

```sh
cd rn-psc-new

yarn install
```

## How to run

_start expo:_

```sh
yarn start
// or
expo start
```

## How to build

```sh
expo build:android
```
